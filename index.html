How it works

Three steps to a clean session

01
Point to your stems folder

Select your stems output folder. DAWganize scans all audio files and detects formats automatically — WAV, AIFF, FLAC.

02
AI groups your tracks

Drums, Bass, Harmonic, Melody, Vocals — classified automatically. Drag to reassign, or let the AI handle everything.

03
Import into your DAW

One click creates and imports a fully organized session into Pro Tools, Ableton, or Logic Pro — with routing, bus sends, and controller names set.

Features

Built for working producers

Every feature was built to solve a real problem in a real session — not a feature list for its own sake.

🧠
AI Instrument Recognition

Local FFT spectral analysis classifies tracks by audio content — Drums, Bass, Vocals, Keys and more. Falls back to keyword matching for MIDI tracks. No cloud API, no subscription.

🔌
Pro Tools I/O Routing

Maps each group to hardware output pairs using the Pro Tools Scripting API. Scans your open Pro Tools session live. Save named profiles and load them instantly on any project.

📂
Stem Import Engine

Scans WAV, AIFF, and FLAC stems. Auto-detects mono/stereo. Creates a complete session file with audio tracks, groups, file references, and timeline placement — for all three DAWs.

🎚️
HUI Controller Names

Condenses track names to 4 characters for HUI and Avid S-Series hardware. Strip Common Prefix, Strip Session Prefix, and Condense Names buttons — one click to a clean console view.

🔍
Stereo-to-Mono Detection

Detects fake stereo files with identical L/R channels and converts them to mono in one click — cleaner mix bus, half the channel count, zero manual checking.

🎨
DAW Color Support

Full Ableton 70-color palette written directly into the .als file. Logic Pro auto-rainbow mode triggers 96-color auto-assign and applies group colors to every Track Stack header automatically.

🎵
Ableton Export Engine

Export MIDI as standard .mid files with polyphonic timing. Audio export with configurable sample rate (48kHz), bit depth (24-bit), format (WAV/AIFF), and per-track mono/stereo override.

🔒
Non-Destructive & Private

Your original session files are never overwritten. All AI runs locally — no data leaves your Mac. No cloud API, no account required. Reviewed for security: no command injection, no path traversal, no data exfiltration.

DAW Support

One tool. Three DAWs.

Ableton Live
.ALS · Live 11 & 12

Full session prep — reads/writes .als files directly (gzipped XML). AI grouping, 70-color palette, strip silence, stereo-to-mono, auto routing, and MIDI/audio export. Auto-detects Live 11 vs 12 schema.

Requires: Ableton Live 11.0+ · macOS 12+ Coming soon
Pro Tools
.AAF · Pro Tools 2023.3+

One-click AAF import with automatic dialog handling. I/O output routing via the Pro Tools Scripting API. HUI 4-char controller names. Reverb/Delay bus sends. Scan from open Pro Tools session.

Requires: Pro Tools 2023.3+ · macOS 12+ Coming soon
Logic Pro
.LOGICX · Logic 10.7.5+

Two import modes: Auto rainbow (96-color batch open) or Colorize yourself (AAF). Creates Summing Track Stacks per group with header colors. Bus sends via embedded Mixer. AppleScript automation.

Requires: Logic Pro 10.7.5+ · macOS 12+ Coming soon
